Voting Analysis Project

Comprehensive Analysis of 2020 U.S. Presidential Election in Georgia and Texas

Project Overview

This project analyzes voting patterns in the 2020 U.S. Presidential Election, focusing on the states of Georgia and Texas. It combines county-level voting data with income and education statistics to explore correlations and visualize key trends. The analysis includes vote percentages, county-level comparisons, correlation matrices, and margin of victory analysis.

Methodology

Data Processing Pipeline

  1. Data Cleaning: Standardized and cleaned raw voting, income, and education datasets
  2. Data Merging: Combined annual CSVs into comprehensive datasets
  3. Analysis: Performed statistical analysis and correlation studies
  4. Visualization: Created interactive charts and graphs using matplotlib and seaborn
